Huawei hat angekündigt, ein eigenes Betriebssystem — Harmony OS. In der Zukunft kann Sie ersetzen auf Android-Smartphones des Unternehmens. Allerdings ist das Betriebssystem konzipiert und für eine Vielzahl von anderen Geräten.

Die neue Plattform, früher bekannt als Hongmeng OS, läuft auf микроядре. Dadurch können Sie auf vielen verschiedenen Geräten — Smart-Spalten und Uhren, Autos, Computern, Tablets und Smartphones. Das Betriebssystem funktioniert genauso gut auf beiden Geräten mit sehr geringem und mit großem Speicher.<figure>

![](http://de.bestwow.net/wp-content/uploads/2019/09/huawei-predstavila-operacionnuyu-sistemu-harmony-os-al-ternativu-android-1.jpg) <figcaption>phonearena.com</figcaption></figure> 

In der Zukunft auf Harmony OS ausgeführt werden können HTML5-, Linux-und Android-Apps. Huawei stellt Entwicklern den Zugriff auf die eigenen Compiler Code Ark, die unterstützt eine Vielzahl von Programmiersprachen. Auch Unternehmen können das neue SDK, um Anwendungen auf verschiedenen Geräten. Aus Sicherheitsgründen sperrt Huawei root-Zugriff auf Harmony OS.

Huawei hat wiederholt, dass will nicht aufgeben Android. Aber wenn das Unternehmen sich in einer ausweglosen Situation, dann wechselt es sofort auf Ihr Betriebssystem. Es dauert maximal ein paar Tage.

Harmony OS ist zum größten Teil für den chinesischen Markt, aber es ist nur noch. Im nächsten Jahr öffnet das Unternehmen den Quellcode des OS, dass Sie verwenden konnten und andere Hersteller.

Das erste Gerät auf der Harmony OS wird ein TV Honor Vision TV. Es geht auf den Verkauf in China schon morgen.
