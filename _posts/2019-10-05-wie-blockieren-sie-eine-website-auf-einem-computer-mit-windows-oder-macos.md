Alle unten aufgeführten Methoden blockieren den Zugang zu ausgewählten Websites über alle Browser. Versierte Computer-Benutzer kann umgehen diese Sperre. Aber für den Umgang mit zaudern und Kontrolle über die Kinder dieser Maßnahmen sollte ausreichen.

## Wie Webseite sperren auf der Ebene des Routers

Viele Router in den Einstellungen können Sie Steuern schwarze Liste von Websites. Zugriff auf freigegebene ihm die URL gesperrt für alle angeschlossenen Geräte mit dem Router über Wi-Fi. Um jede Webseite entsperren, genügt es, löschen Sie seine Adresse aus der Liste.

Überprüfen Sie, ob Ihr Router gesperrt werden. Dafür gehen Sie in Ihre Einstellungen: geben Sie im Browser 192.168.0.1 oder eine andere Adresse, die auf der Unterseite des Routers, und geben Sie Benutzernamen und Passwort für die Anmeldung. Dann suchen Sie den Abschnitt «Filter», oder «Zugangssteuerung», oder andere in der Nähe von im Sinne von Titel.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-1.jpg) 

Wenn Sie das Menü mit den Einstellungen der schwarzen Liste, öffnen Sie Sie und fügen Sie die Adressen unerwünschter Seiten. Herum sollte Tipps, die Ihnen helfen zu verstehen.

Dies ist der Schnellste Weg, um beschränken Sie den Zugriff auf Internet-Ressourcen sofort für die gesamte Wi-Fi-Technik in den Raum. Aber wenn Sie eine Website sperren möchten nur für ausgewählte Geräte oder auch einzelne Benutzer, werfen Sie einen Blick auf die folgenden Optionen.

## Wie eine Website blockiert in Windows

### 1. Mit Hilfe von hosts-Datei

Auf jedem Computer unter Windows eine TXT-Datei mit dem Namen hosts. Wenn Sie es fügen Sie alle URLS, Browser nicht in der Lage, öffnen Sie die entsprechenden Seiten.

So Bearbeiten Sie die betreffende Datei zuerst öffnen Sie den Editor: klicken Sie mit der rechten Maustaste auf das Programm-Icon und wählen Sie «als Administrator Ausführen». In einem Editor-Fenster klicken Sie auf «Datei» → «Öffnen», ändern Sie den Anzeigemodus mit «Word-Dokumente» auf «Alle Dateien» und wählen Sie die Hostdatei, die sich unter C:WindowsSystem32driversetc.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-2.jpg) 

Wenn die hosts-Datei, tiefer Text, und fügen Sie eine oder mehrere Adressen von Websites, die Sie blockieren möchten, im folgenden Format ein: 127.0.0.1 URL. Nach zahlen unbedingt Leerzeichen, anstelle der URL die URL einer Website geben, ohne Teile _https://_. Wenn Sie mehrere Einträge hinzufügen, jede in einer neuen Zeile eingeben. Beispiel:

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-3.jpg) 

Nach dem hinzufügen von Adressen speichern Sie die Datei und starten Sie den Computer neu, um die änderungen zu übernehmen.

Wenn Sie möchten, um Webseiten zu entsperren, wieder öffnen Sie die hosts-Datei und löschen Sie Ihre Aufnahmen.

### 2. Durch spezielle Software

Wenn Sie wollen, sperren Sie ablenkende Websites, um nicht прокрастинировать, achten Sie auf das Kostenlose Tool Cold Turkey. Sie können hinzufügen einige URLS und die Zeit zu bestimmen, auf welche Sie nicht auch auf Ihrem Computer. Solange diese Frist nicht abgelaufen ist, werden Sie nicht in der Lage, die Sperre wieder aufzuheben.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-4.jpg) 

Bei Cold Turkey es gibt auch eine kostenpflichtige Version, die blockieren können nicht nur Webseiten, sondern auch ablenkende App.

### 3. Mit Hilfe der Kindersicherung

Diese Methode eignet sich speziell für Eltern, die wollen Ihre Kinder vor Pornografie und anderen Inhalten für Erwachsene. Mit Hilfe der Kindersicherung von Windows können Sie eine Liste der Websites erstellen, die blockiert werden sollen, und aktivieren Sie die automatische Sperrung aller Websites für Erwachsene. Alle Einschränkungen gelten nur für das Profil eines Kindes und betreffen nicht Ihr Konto.

Wir betrachten das sperren von Websites für Kinder-Profils am Beispiel von Windows 10. In früheren Versionen von Verfahren kann variieren, aber die Allgemeine Vorgehensweise ähnlich.

Gehen Sie zu «Start» → «Einstellungen» → «Accounts» → «persönliche Daten» und stellen Sie sicher, dass Sie sich angemeldet haben, durch das Microsoft-Konto: auf dem Bildschirm angezeigt werden soll Ihre E-Mail. Wenn nicht, registrieren Sie Ihr Konto.

Weiter, wählen Sie in der Seitenleiste «Familie und andere Leute» und klicken Sie auf «Familienmitglied Hinzufügen». Im nächsten Fenster fügen Sie das Konto des Kindes mit Eingabeaufforderungen des Systems. Im Prozess müssen Sie durch den Anschluss einer beliebigen Mailbox, öffnen Sie es und bestätigen die Registrierung des neuen Profils.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-5.jpg) 

Dann kehren Sie zum Menü «Familie und andere Leute» und klicken Sie auf «Verwalten familiären Einstellungen über das Internet».

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-6.jpg) 

Im Gegensatz zu älteren OS-Versionen, wo die profile liegen alle Parameter in einem Abschnitt, die weitere Konfiguration von Windows 10 gibt es auf der Microsoft-Website. Wenn die Webseite, melden Sie sich an Ihrem Konto an und öffnen Sie den Bereich «Familie». Sah das Profil Ihres Kindes aus, klicken Sie daneben auf «Beschränkung auf Inhalte».

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-7.jpg) 

Scrollen Sie nach unten zum Abschnitt «Browsen». Hier können Sie das automatische sperren von Webseiten mit dem Schalter «Blockieren ungültige Web-Sites» und fügen Sie manuell Ihre Webseiten in die Liste «Immer verweigern».

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-8.jpg) 

Diese Einschränkungen gelten nur für die Browser Microsoft Edge und Internet Explorer. Die anderen Browser im Kinder-Profil wird komplett gesperrt.

## Wie blockieren Sie eine Website in macOS

### 1. Mit Hilfe von hosts-Datei

macOS wie Windows, blockiert die Adressen der Websites, die Anwender in die System-hosts-Datei. Sie müssen nur öffnen Sie diese Datei und fügen Sie die unerwünschte URL. Browser nicht öffnen, solange Sie nicht entfernen Sie diese Adressen aus der hosts-Datei.

Zuerst öffnen Sie die betreffende Datei. Dazu starten Sie das Dienstprogramm «Terminal» (Finder → «Programme» → «Dienstprogramme» → «Terminal»), geben Sie den Befehl _sudo /bin/cp /etc/hosts /etc/hosts-original_ und drücken Sie die EINGABETASTE. Wenn Sie aufgefordert werden, das Passwort Ihres Kontos, geben Sie es und drücken Sie erneut die EINGABETASTE. Dann geben Sie den Befehl _sudo nano /etc/hosts_ und wieder drücken Sie die EINGABETASTE. Wenn Sie aufgefordert werden, geben Sie Ihr Passwort erneut.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-9.jpg) 

Wenn die hosts-Datei, tiefer Text, und fügen Sie eine oder mehrere Adressen von Websites, die Sie blockieren möchten, im folgenden Format ein: 127.0.0.1 URL. Nach zahlen unbedingt Leerzeichen, anstelle der URL die URL einer Website geben, ohne Teile _https://_. Wenn Sie mehrere Einträge hinzufügen, jede in einer neuen Zeile eingeben. Beispiel:

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-10.jpg) 

Indem Sie alle erforderlichen Adressen, drücken Sie Strg + X, um aus der hosts-Datei. Starten Sie den Computer neu, damit die änderungen wirksam werden.

Wenn Sie möchten, um Webseiten zu entsperren, wieder öffnen Sie die hosts-Datei und löschen Sie Ihre Aufnahmen.

### 2. Durch spezielle Software

Ein kostenloses und sehr einfaches Programm SelfControl ermöglicht das blockieren alle Websites für den gewünschten Zeitraum sind. Solange es nicht abgelaufen ist, können Sie nicht öffnen Sie im Browser, auch wenn Sie ein Programm deinstallieren oder starten Sie Ihren Mac. Eine gute Möglichkeit für den Umgang mit Ablenkungen Netzwerkressourcen.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-11.jpg) 

Darüber hinaus, die oben genannte Cold Turkey, die funktioniert in ähnlicher Weise, auch in einer Version für macOS.

### 3. Mit Hilfe der Kindersicherung

Der Mechanismus der Kindersicherung in macOS ermöglicht den Zugriff auf ausgewählte Standorte nur für einen einzigen Benutzer auf dem Computer. Zum Beispiel, Ihres Kindes. Andere Benutzer können auf alle Websites gehen ohne Einschränkungen.

Um die Sperre einrichten, öffnen Sie das Apple-Menü und gehen Sie zu «Systemeinstellungen» → «Kindersicherung». Fügen Sie ein neues Konto für Ihr Kind.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-12.jpg) 

Nach klicken Sie auf den hinzugefügten Profil und klicken Sie auf die Registerkarte «Web». Hier können Sie wählen, welche Seiten stehen dem Kind unter seinem Konto und welche nicht.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-zablokirovat-sayt-na-komp-yutere-s-windows-ili-macos-13.jpg)
