Angenommen, Sie müssen Sie entfernen irgendwoher getippten Text. Aus einem geschützten PDF, mit Bild, mit einem Screenshot, von der Seite, wo das kopieren des Textes ist deaktiviert — ja von überall her. Sie können sicher installieren sperrige und leistungsstarke ABBYY FineReader, aber in den meisten Fällen seine Möglichkeiten Redundant sind. Ein kleines Dienstprogramm Easy Screen OCR erkennt den Text schneller als Fine Reader gestartet.

Downloaden und installieren Sie Easy Screen OCR. Die Anwendung hat eine Version für Windows und macOS. Nach der Installation und dem Start in der Taskleiste Ihres Betriebssystems erscheint das Symbol. Klicken Sie mit der rechten Maustaste und öffnen Sie die Einstellungen (Preferences).

![](https://de.bestwow.net/wp-content/uploads/2019/09/easy-screen-ocr-dlya-windows-i-macos-raspoznaet-tekst-s-kartinok-ili-pryamo-s-ekrana-1.jpg) 

In den Einstellungen können Sie aktivieren, starten Sie das Programm zusammen mit dem System. Außerdem auf der Registerkarte «Sprache» (Language) ist eine wichtige Option — Dokumentensprache.

![](https://de.bestwow.net/wp-content/uploads/2019/09/easy-screen-ocr-dlya-windows-i-macos-raspoznaet-tekst-s-kartinok-ili-pryamo-s-ekrana-2.jpg) 

Wechseln Sie zu diesem Register, wählen Sie in der Dropdown-Liste die OCR Languages Deutsch, und schließen Sie die Einstellungen. Nur Easy Screen OCR unterstützt etwa hundert Sprachen.

Jetzt ist das Programm betriebsbereit. Um zu erkennen beliebigen Text auf dem Bildschirm, klicken Sie auf das Symbol Easy Screen OCR in der Taskleiste mit der rechten Maustaste, und wählen Sie Capture. Sie können wählen Sie einen Bereich des Bildschirms mit некопируемым Text.

![](https://de.bestwow.net/wp-content/uploads/2019/09/easy-screen-ocr-dlya-windows-i-macos-raspoznaet-tekst-s-kartinok-ili-pryamo-s-ekrana-3.jpg) 

Dann unten erscheint ein Popup-Fenster mit dem erfassten Bereich. Klicken Sie auf die Schaltfläche OCR, und das Programm zeigt Ihnen den fertigen Text. Können Sie es kopieren und Bearbeiten.

![](https://de.bestwow.net/wp-content/uploads/2019/09/easy-screen-ocr-dlya-windows-i-macos-raspoznaet-tekst-s-kartinok-ili-pryamo-s-ekrana-4.jpg) 

Darüber hinaus Easy Screen OCR ist in der Lage, das kopieren von Text in Bildern. Wählen Sie dazu im Menü Image OCR und ziehen Sie das gewünschte Bild in das angezeigte Fenster. Das Programm kann Scannen nicht nur gedruckte Texte, sondern auch handschriftliche.

![](https://de.bestwow.net/wp-content/uploads/2019/09/easy-screen-ocr-dlya-windows-i-macos-raspoznaet-tekst-s-kartinok-ili-pryamo-s-ekrana-5.jpg)
