## 1. Fügen Sie die Verknüpfung für den Start

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-ispol-zovat-gmail-v-kachestve-desktopnogo-prilozheniya-1.jpg) 

So öffnen Sie Gmail als eine normale Anwendung erstellen Sie eine Verknüpfung und legen Sie es auf Ihrem Desktop, Taskleiste oder im Dock. Das geht so:

  * Öffnen Sie Gmail in Chrome.
  * Gehen Sie zu «Menü» → «Weitere Tools» und wählen Sie «Verknüpfung Erstellen».
  * Setze ein Häkchen bei «Öffnen in einem separaten Fenster» und klicken Sie auf «Erstellen».
  * In Windows die Verknüpfung wird auf dem Desktop, macOS — im Launchpad.

## 2. Schalten Sie den offline-Modus

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-ispol-zovat-gmail-v-kachestve-desktopnogo-prilozheniya-2.jpg) 

Mal Gmail App jetzt und nicht Online-Preise, es sollte auch ohne Internet funktionieren. Dazu aktivieren offline-Modus:

  * Öffnen Sie im Ordner «Posteingang», klicken Sie auf das ZAHNRAD-Symbol und wählen Sie «Einstellungen».
  * Klicken Sie auf die Registerkarte «Offline» und setze ein Häkchen bei «Aktivieren offline-Zugriff auf E-Mails».
  * Klicken Sie auf «änderungen Speichern» unten auf der Seite.

## 3. Stellen Sie Google Mail als Haupt-Client

Um bei der Verarbeitung von E-Mail-System Links nicht перебрасывала in der Standard-Client, müssen Sie Gmail als E-Mail-Anwendung standardmäßig. Unter Windows und macOS dies geschieht auf unterschiedliche Weise.

### Windows

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-ispol-zovat-gmail-v-kachestve-desktopnogo-prilozheniya-3.jpg) 

  * Starten Sie das Menü «Start» und klicken Sie auf das kleine ZAHNRAD-Symbol.
  * Navigieren Sie zu «Anwendungen» → «Standard-Apps».
  * Wählen Sie als primäre E-Mail-Programm Google Chrome.

### macOS

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-ispol-zovat-gmail-v-kachestve-desktopnogo-prilozheniya-4.jpg) 

  * Öffnen Sie die Anwendung «Mail».
  * Navigieren Sie zu «Einstellungen» → «Allgemein».
  * Aus der Dropdown-Liste «E-Mail-Client» wählen Sie Google Mail.app.

## 4. Aktivieren Sie Benachrichtigungen

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-ispol-zovat-gmail-v-kachestve-desktopnogo-prilozheniya-5.jpg) 

Ohne Benachrichtigungen über neue E-Mails E-Mail-Client wäre nicht möglich. Google Mail unterstützt diese Funktion und ist in der Lage Benachrichtigungen anzeigen auf dem Desktop wie alle Anwendungen. Konfigurierbar ist wie folgt.

  * Öffnen Sie den «Posteingang», klicken Sie auf das kleine ZAHNRAD-Symbol und wählen Sie «Einstellungen».
  * Auf der Registerkarte «Allgemein» klicken Sie im Abschnitt «Desktop-Benachrichtigungen» und aktivieren Sie «Aktivieren Sie die Benachrichtigung über neue E-Mails».
  * Klicken Sie auf die Schaltfläche «änderungen Speichern» unten auf der Seite.

## 5. Verwenden Sie Hotkeys

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-ispol-zovat-gmail-v-kachestve-desktopnogo-prilozheniya-6.jpg) 

Hilfreich großartig beschleunigen die Arbeit, so dass Sie schnell zu bereinigen Schutt-Mails und E-Mails beantworten. Wie die Online-Version von Gmail Desktop Anwendung unterstützt auch Sie. Alles was Sie tun müssen, ist die Funktion in den Einstellungen aktivieren.

  * Auf dem Bildschirm «Eingehende» klicken Sie auf das Zahnradsymbol und öffnen Sie die «Einstellungen».
  * Klicken Sie auf die Registerkarte «Allgemein» und suchen Sie den Abschnitt «Бычтрые Tasten».
  * Markieren Sie die Option «Aktivieren» und klicken Sie auf «änderungen Speichern» am unteren Rand der Seite.
