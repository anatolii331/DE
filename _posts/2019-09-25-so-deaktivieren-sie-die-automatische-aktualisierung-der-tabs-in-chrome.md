Zuvor die entsprechende Funktion man deaktivieren unter versteckten Einstellungen des Browsers, aber in Chrome 75.0 jeden Euro Wert. Jetzt nicht mehr verbieten automatische Updates geöffneten Tabs von den standardmäßigen Mitteln, aber Sie können dies tun mit einer speziellen Erweiterung.

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-otklyuchit-avtoobnovlenie-vkladok-v-chrome-3.jpg) 

  * Klicken Sie auf den Link in den Chrome Web Store.
  * Klicken Sie auf die Schaltfläche «Installieren» und dann auf «Erweiterung Installieren».
  * Aktivieren Sie das Plugin, indem Sie auf das Symbol in der Menüleiste.
  * Fertig!

Sicherstellen, dass die Tabs nicht mehr neu gestartet werden indem man in der Adresszeile `chrome://discards` und schaute Whisker Auto Discardable — gegenüber jeder Seite dort soll Kreuz stehen.

![](http://de.bestwow.net/wp-content/uploads/2019/09/kak-otklyuchit-avtoobnovlenie-vkladok-v-chrome-4.jpg) 

Vergessen Sie nicht, dass die automatische Aktualisierung deaktivieren, können Sie nicht sehen, die letzten änderungen im Inhalt des alten Tabs. Darüber hinaus verwendet der Browser mehr Systemressourcen, was die Leistung reduzieren kann auf langsameren Computern.
