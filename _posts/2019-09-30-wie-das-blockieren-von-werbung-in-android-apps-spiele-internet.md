Ich will sofort bemerken, dass Werbung ein notwendiges übel. Die Macher Programme bieten uns die Kostenlose Nutzung Ihrer Produkte in der Hoffnung, dass wenigstens die bescheidenen Werbeeinnahmen make-up für von Ihnen aufgewendeten Zeit und Mühe. Aber die Werbung blockiert werden kann. Ich, zum Beispiel, auf dem Tablet des Kindes vollständig blockiert Werbeanzeigen, um nicht abgelenkt endlose Angebote spielen in einem neuen Spiel oder Rezepte Brustvergrößerung auf Web-Seiten. In diesem Artikel erfahren Sie, wie Sie dies tun auf jedem Smartphone oder Tablet mit Android.</p> 

## Adblock Plus

Ein beliebtes add-on für Browser hat eine Version für Android, aber Sie finden es nicht im Google Play Store aufgrund von Beschränkungen von Google. Allerdings können Sie ihn als eigenständige Anwendung, indem Sie auf dieser Seite. Vor der Installation der Software stellen Sie sicher, dass Sie zum installieren von unbekannten Quellen.

Das Programm arbeitet sowohl auf рутированных Geräten, als auch auf herkömmliche, hat aber einige Unterschiede. Im ersten Fall (Ruth) benötigen Sie root-Zugriff zu gewähren, und genießen können. Im zweiten Fall müssen Sie ein paar zusätzliche Schritte zum einrichten, deren Sequenz ausführlich hier (engl.). Danach Adblock Plus beginnt, Werbung zu blockieren. Sie können einige Optionen für seine Arbeit in den Einstellungen.

![](https://de.bestwow.net/wp-content/uploads/2019/09/kak-blokirovat-reklamu-v-android-prilozheniya-igry-internet-1.jpg) 

Wenn Sie sperren wollen nur Werbung in Ihrem Browser, dann müssen Sie einen genaueren Blick auf die Mobile Version von Firefox, weil Adblock Plus hat die Erweiterung nur für diesen Browser. Download Firefox Mobile können Sie hier, und öffnen Sie darin die Seite der Erweiterung und legen Sie es auf die übliche Weise. Starten Sie Ihren Browser, um loszulegen Erweiterungen.

![](https://de.bestwow.net/wp-content/uploads/2019/09/kak-blokirovat-reklamu-v-android-prilozheniya-igry-internet-2.png) 

## Werbeblocker mit der hosts-Datei

Diese Methode verwendet eine spezielle Datei &#8220;hosts&#8221; (was ist das?), um Werbung zu blocken. Dies gilt für alle Banner in Ihrem Gerät, einschließlich der in verschiedenen Anwendungen und spielen. Gilt nur auf рутированных Geräten.

Die richtige hosts-Datei mit verordneten Adressen der wichtigsten Banner-Netzwerke können auf diesen Seiten (eins, zwei und drei). Klicken Sie einfach auf den Link und speichern Sie den Inhalt als Text-Datei mit dem Namen **hosts**.

Über eine für Sie geeignete Methode переправьте diese Datei auf Ihr mobiles Gerät. Dann öffnen Sie den Dateimanager, der root-Zugriff hat, zum Beispiel Root Explorer, und speichern Sie die Datei in /**etc** oder **/system/etc.**

![](https://de.bestwow.net/wp-content/uploads/2019/09/kak-blokirovat-reklamu-v-android-prilozheniya-igry-internet-3.jpg) 

Sehr vernünftig wird vorab die ursprüngliche Version der Datei speichern, indem Sie ihm einfach ein anderer name. Dies ist nützlich, wenn etwas schief geht oder Sie wollen, dass alles wieder wie früher. Starten Sie Ihr Gerät und genießen.
