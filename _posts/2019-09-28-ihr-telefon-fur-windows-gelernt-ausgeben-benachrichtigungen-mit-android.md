Die App «Telefon» für Windows 10 kann nun Benachrichtigungen von Ihrem Android-Smartphone. Sie können Nachrichten von Telegram, WhatsApp und andere Apps direkt auf Ihrem Computer. Wenn Sie die Benachrichtigung mit dem PC wird verschwinden und mit Telefon.

Solange die Möglichkeit auf eine Nachricht zu Antworten Nein. Aber die Website Thurrott herausgefunden, dass es später erscheint. Ein Klick auf die Meldung auf dem Computer erscheint der Bildschirm des Smartphones, und Sie können tun alle das gleiche, was Sie tun direkt auf Ihrem mobilen Gerät.<figure>

![](https://de.bestwow.net/wp-content/uploads/2019/09/-1.jpg) <figcaption>thurrott.com</figcaption></figure> 

Anzeigen von Benachrichtigungen auf dem PC verfügbar sein Abonnenten-Programm Windows Insider April 10. Bis zum Ende der Woche die Funktion wird für alle Benutzer des Betriebssystems. Die Hauptsache ist, dass Sie das Update mindestens von April 2018.

Neben anderen Funktionen «des Telefons» — die Möglichkeit, eine Galerie von Bildern mit einem Smartphone, sondern auch Lesen und beantworten von SMS-Nachrichten.
