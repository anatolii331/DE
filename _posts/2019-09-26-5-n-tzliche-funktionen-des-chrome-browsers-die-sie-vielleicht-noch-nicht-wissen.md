### Eine Verknüpfung zu einer Seite auf dem Desktop

Wenn Sie sehr viele Lesezeichen im Browser, und Sie speichern, musst du noch ein paar Links, verwenden Sie die Option Verknüpfung. Mit seiner Hilfe können Sie senden Sie auf dem Desktop das Icon der Webseite, перенаправляющую sofort auf die gewünschte Seite.

![](http://de.bestwow.net/wp-content/uploads/2019/09/5-poleznyh-funkciy-brauzera-chrome-o-kotoryh-vy-mogli-ne-znat-1.jpg) 

Für die auf der gewählten Seite öffnen Sie die Chrome-Einstellungen und wählen Sie «Weitere Tools». Im Untermenü wird die Option Verknüpfung erstellen. Klicken Sie darauf und geben Sie den Link-Titel.

### Speichern von Seiten in PDF

Wenn Sie brauchen, speichern Sie die Seite genau in der Form, in der es im Browser angezeigt wird, können Sie die Option speichern im PDF-Format. Es ist erreichbar über das Menü drucken. Um es aufzurufen, drücken Sie Ctrl / Cmd + P und auf der linken Seite wählen Sie die Option «als PDF Speichern».

Verderben PDF-Datei können, es sei denn, die Werbe-Banner. Es wird in der Vorschau sichtbar.

### Anzeigen einer zwischengespeicherten Version der Website

![](http://de.bestwow.net/wp-content/uploads/2019/09/5-poleznyh-funkciy-brauzera-chrome-o-kotoryh-vy-mogli-ne-znat-2.jpg) 

Zum wiederherstellen einer früheren Version einer Webseite oder öffnen Sie eine bereits gelöschte Artikel von der Website, können Sie auf den Cache zugreifen. Dazu vor der Adresse der Seite hinzufügen möchten «cache:» und drücken Sie «Enter». Auf diese Weise kann man Sie vorher aufgerufenen Seiten ohne Verbindung zum Netzwerk.

### Rückerstattung geschlossenen Tab mit der Tastenkombination

Öffnen, die zuletzt geschlossene Registerkarte gibt es mehrere Möglichkeiten. Der unangenehmste — eine Seite in der Browser-History. Etwas praktischer — mit der rechten Maustaste auf einen anderen Tab und in der Drop-down-Menü wählen Sie «geschlossenen Tab wieder Öffnen».

Die noch schneller Weg — die Kombination der Tasten Ctrl / Cmd + Shift + T.

### YouTube-Videos können im Modus «Bild in Bild»

![](http://de.bestwow.net/wp-content/uploads/2019/09/5-poleznyh-funkciy-brauzera-chrome-o-kotoryh-vy-mogli-ne-znat-3.jpg) 

Um die arbeiten im Browser oder anderen Programmen ohne loslsung von YouTube-Video kann in einem separaten Fenster öffnen, wird die Oberseite der alle Ihre Anwendungen angezeigt werden. Dazu navigieren Sie zum gewünschten Video, klicken Sie mit der rechten Maustaste, öffnen Sie das erste Menü &#8220;Grundeinstellungen&#8221;, und klicken Sie dann hier ein zweites mal. Öffnet sich ein anderes Kontextmenü, in dem Sie den Punkt «Bild in Bild».

Diese Funktion versetzt die Walze in eine kleine Box, die bleibt auf dem Bildschirm, selbst wenn Sie Rollen den Browser. Maße des Fensters kann eingestellt werden, dehnen Sie es an den Kanten in verschiedene Richtungen.
