## Laden Sie die Distribution

Zu Beginn ist Android keine Unterstützung von Computern mit x86-Architektur, und die alten Versionen waren ausschließlich für Mobile Geräte. Jetzt gibt es keine Einschränkungen. Auf der Entwickler-Website des Projektes Android-x86 Download-Distribution-System, das ohne Probleme auf jedem Computer installiert. Die aktuellste Version für die Installation — Android 7.1.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-1.jpg) 

Bei der Auswahl der Distribution achten Sie auf die Bitanzahl. Wenn auf dem Computer die x86 Architektur verwendet, müssen Sie das entsprechende Installationspaket. Ansicht die Art des Systems können Sie in den Eigenschaften von «Computer».

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-2.jpg) 

## Um das Installationsmedium zu erstellen

Für die Erstellung bootfähiger Medien empfohlen einen USB-Stick mit einer Kapazität von mindestens 2 GB. Dateisystem — FAT32.

Für die Aufnahme der Distribution auf einem USB-Stick verwenden Sie das Programm Rufus. Schließen Sie das Laufwerk an den Computer an und setzen Sie in Rufus Einstellungen, wie im Screenshot zu sehen. Wählen Sie erstellen von ISO-Image und klicken Sie auf das Laufwerk, um den Pfad скачанному Datei Android.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-3.jpg) 

Bei der Frage über die Art der Aufnahme wählen Sie ISO. Während der Erstellung bootfähiger Medien mit USB-Stick werden alle Daten gelöscht.

## Starten Sie ohne Installation

Bei der Distribution von Android-x86 gibt es eine Option zum starten des Systems ohne Installation. In diesem Modus können Sie alle Funktionen von Android, aber die änderungen werden nicht gespeichert. In der Tat, dies ist der Demo-Modus, das hilft, zu verstehen, ob es zweckmäßig ist, funktioniert mit Android auf dem Computer.

Starten Sie den Computer neu und Booten Sie vom USB-Stick — genauso wie bei der Neuinstallation von Windows. Im Menü wählen Sie die erste Option Run Android without installation.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-4.jpg) 

Nach einem kurzen Download der Bildschirm für die Sprachauswahl und andere Grundeinstellungen. In dieser Phase sollte funktionieren Tastatur, Maus und Touchpad, wenn Sie eine auf Android Laptop. Kann man nichts anpassen — immer noch in diesem Modus wird die Konfiguration nicht gespeichert.

Drücken Sie «Weiter», bis Sie gelangen zum Startbildschirm des Systems. Überprüfen Sie, wie funktioniert diese Version von Android auf dem PC. Wi-Fi, LAN, WLAN, Videowiedergabe — alles muss funktionieren standardmäßig, ohne weitere Einstellungen.

## Installieren Sie das System

Wenn Sie möchten, um auf dem Computer arbeiteten zwei Systeme, erstellen Sie eine Partition für die Installation von Android. Installation in eine Partition mit Windows führen alle aktuellen Systeme. Unter Android sollte mindestens 8 GB freien Speicherplatz. Die optimale Größe von 16 GB.

Klicken Sie im Installationsfenster auf den letzten Punkt Install Android to harddisk. Wählen Sie den markierten Abschnitt unter Android.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-5.jpg) 

Geben Sie das Dateisystem FAT32 oder NTFS. Es erscheint eine Warnung über die Zerstörung aller Daten von der ausgewählten Partition. Drücken Sie Yes.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-6.jpg) 

Nehmen Sie das Angebot den GRUB-Bootloader installieren. Legen Sie den Unterschlüssel EFI GRUB2, wenn die Installation von Android auf dem PC mit UEFI. Wenn es das übliche BIOS, drücken Sie Skip.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-7.jpg) 

Klicken Sie auf ja im Dialog «Do you want to install/system directory as read-write?», um überschreiben der Daten im System.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-8.jpg) 

Warten Sie, bis die Installation abgeschlossen. Nach der Installation aufgefordert werden, erstellen Sie eine benutzerdefinierte Partition innerhalb des Systems. Klicken Sie auf Yes, und installieren Sie die Größe des 2 000 MB.

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-ustanovit-android-na-komp-yuter-9.jpg) 

Klicken Sie auf Run Android, um die Installation abzuschließen. Ersteinrichtung des Systems wiederholt die erste Inbetriebnahme eines neuen Android-Geräte: Sie müssen die Sprache auswählen, Wi-Fi und Google-Konto hinzufügen.

Android-x86 ist nicht die einzige десктопная Android-Version. Zur Abwechslung können Sie setzen Remix OS. Sie positioniert sich als mehr als einfach zu installieren und zu erlernende System, das entwickelt wurde, um ein Computer verwendet, obwohl im Grunde das Android-ökosystem.
