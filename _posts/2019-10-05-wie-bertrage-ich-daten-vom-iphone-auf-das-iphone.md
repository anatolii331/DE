## Schritt 1. Speichern Sie die Daten

Erstellen einer Sicherungskopie Ihrer Informationen können über iCloud oder über iTunes auf Ihrem Computer. Später für die übertragung von Daten verwendet wird, wird die entsprechende Kopie.

### So sichern Sie Ihre iCloud

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-perenesti-dannye-s-iphone-na-iphone-1.jpeg)![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-perenesti-dannye-s-iphone-na-iphone-2.jpeg) 

  1. Öffnen Sie die «Einstellungen» auf Ihrem iPhone und Tippen Sie auf das Symbol Ihres Kontos.
  2. Gehen Sie zu iCloud → «Sichern».
  3. Schalten Sie den gleichnamigen Kippschalter und klicken Sie auf «sichern».

  4. Wenn Sie die Daten speichern möchten, die Programme «Gesundheit» und «Aktivität», setze ein Häkchen bei «Backup Verschlüsseln» und legen Sie ein Passwort. Vergessen Sie nicht den Code, da sonst die Informationen wiederherzustellen!

## Schritt 2. Befreien Sie Apple Watch

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-perenesti-dannye-s-iphone-na-iphone-5.jpeg)![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-perenesti-dannye-s-iphone-na-iphone-6.jpeg) 

Wenn die Uhr iPhone angeschlossen, das für die Bindung an das neue Smartphone müssen Sie zuerst entkoppeln mit dem alten.

  1. Öffnen Sie das Programm Apple Watch.
  2. Klicken Sie auf die Registerkarte «Meine Uhr» und klicken Sie auf das Symbol ℹ️.
  3. Wählen Sie «entkoppeln mit Apple Watch» und bestätigen Sie den Vorgang.

## Schritt 3. Übertragen Sie die Informationen<figure>

![](https://de.bestwow.net/wp-content/uploads/2019/10/kak-perenesti-dannye-s-iphone-na-iphone-7.jpg) <figcaption>igromania.ru</figcaption></figure> 

Importieren Sie können Daten auf drei Arten. Für jeden benötigen eine vorab erstellte Sicherung, aber der Prozess der Migration wird ein wenig anders sein. In einem Fall unbedingt mit einem alten iPhone, in der anderen Computer mit iTunes, und in der Dritten genug Internet-Verbindung.

### 1. Verwenden der Funktion «Quick Start»

<div>
</div>

_YouTube-Kanal Mac Time_

  1. Wenn das neue iPhone bereits aktiviert ist, dann entfernen Sie zuerst alle Daten, indem Sie unter «Einstellungen» → «Allgemein» → «Zurücksetzen» → «Inhalte & Einstellungen Löschen». Wenn nicht, legen Sie die SIM-Karte im Smartphone und Folgen Sie den Anweisungen des Konfigurationsassistenten.
  2. Schalten Sie ein und legen Sie beide iPhone in der Nähe. Auf dem neuen Gerät erscheint der Bildschirm «Quick Start», und auf dem alten — «neues iPhone konfigurieren».
  3. Klicken Sie auf «Weiter» auf dem alten Smartphone und entsperren Sie es.
  4. Warten Sie, bis die Animation auf dem neuen iPhone und bewegen Sie die Kamera alten.
  5. Warten Sie einige Sekunden und schließen Sie die Konfiguration auf das neue Gerät.
  6. Geben Sie das Passwort von der Apple ID und Folgen Sie den Anweisungen.
  7. Auf dem Bildschirm «Apps und Daten» wählen Sie «von iCloud Wiederherstellen».

### 2. Wiederherstellen aus iCloud-Backup

<div>
</div>

_YouTube-Kanal Mac Time_

  1. Wenn das neue iPhone bereits aktiviert ist, dann entfernen Sie zuerst alle Daten, indem Sie unter «Einstellungen» → «Allgemein» → «Zurücksetzen» → «Inhalte & Einstellungen Löschen». Wenn nicht, legen Sie die SIM-Karte im Smartphone und Folgen Sie den Anweisungen des Konfigurationsassistenten.
  2. Auf dem Bildschirm «Apps und Daten» wählen Sie «aus iCloud-Backup Wiederherstellen».
  3. Melden Sie sich in Ihrem Apple ID Account.
  4. Wählen Sie das gewünschte Backup aus der Liste, die sich nach Datum und Größe.
  5. Warten Sie, während die Wiederherstellung und beenden Sie die Konfiguration.
