Computer-Sicherheitsexperten Jonathan Лейтшу (Jonathan Leitschuh) erzählte von Ihnen gefundenen Schwachstellen in einer Anwendung für Videokonferenzen Zoom. Dieses beliebte auf macOS Service kann ein Schlupfloch für die Verfolgung von Nutzern von Apple-Computern.

Das Problem liegt in der architektonischen Schwachstellen Zoom. Zur Verbesserung der Interaktion mit dem Benutzer erstellt das Programm auf dem PC einen lokalen Server. Er ist eine mögliche Quelle der Bedrohung. Angreifer können Zugriff aus der Ferne.

In der Tat, diese Sicherheitsanfälligkeit ermöglicht das aktivieren der Webcam des Computers erzwingen. Darüber hinaus erstellte Server verwendet werden kann auch nach der Entfernung von Zoom. Benutzer müssen nur klicken auf den Link-Einladung in einer Videokonferenz.

Лейтшу erzählte über das Problem der Entwickler von Zoom bereits im März gewarnt, dass eine veröffentlicht Daten über Sie über 90 Tage. Jedoch, gemäß dem Experten, Vertreter der Gesellschaft ausgestiegen mit der übergangslösung, die er bot Ihnen an, am Anfang. Vollständig das Problem behebt es nicht.<figure>

![](https://de.bestwow.net/wp-content/uploads/2019/09/uyazvimost-zoom-dlya-macos-pozvolyaet-sledit-za-pol-zovatelyami-cherez-veb-kameru-obnovleno-1.jpg) <figcaption>deaktivieren der automatischen Aktivierung der Web-Kamera bei einem eingehenden Einladung im Zoom</figcaption></figure> 

Vertreter Zoom Hinzugefügt, dass lösen das Problem mit der Juli-Update. Bislang schützen sich vor überwachung kann durch deaktivieren in den Einstellungen die automatische Aktivierung der Webcam bei eingehenden Einladung in Videokonferenz (Bild oben).

**Update 1**: Zoom produziert einen speziellen Patch, entdeckte Sicherheitslücke schließen. Er entfernt mit Computern lokale Benutzer des Servers, die Ursache des Problems. Darüber berichtet in einer Erklärung auf der offiziellen Website Zoom. Alle Nutzer des Dienstes jetzt müssen Sie einfach aktualisieren Sie Ihre Client.

**Update 2**: TechCrunch berichtet, dass Apple selbst entschieden, wiederversichert zu werden, die Freigabe Spezial-Patch für macOS, das behebt die Sicherheitsanfälligkeit Zoom. Das Update wird automatisch installiert und löscht den lokalen Web-Server-Anwendung. Aktionen seitens der Benutzer ist nicht erforderlich.
